数据库按分类排序及百分比

发表于 2019-11-03
分类：

· 次浏览

where条件在两个最里面的 from后，都要加上，否则百分比不准确

四舍五入：round 函数

末尾加百分号：concat(num, ‘%’)

中文按拼音排序：

Oracle用函数 nlssort

MySQL：
编码为latin1_swedish_ci：order by birary(字段名) asc
编码为 utf8_general_ci：ORDER BY CONVERT( 字段名 USING gbk ) COLLATE gbk_chinese_ci ASC

自定义枚举类型排序：
order by field( name, ‘value1’, ‘value2’ )

示例：研究土壤对生长势的影响

显示结果：
